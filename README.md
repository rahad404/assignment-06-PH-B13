# DigiTools

A modern, responsive digital tools marketplace application built with React and Tailwind CSS. Browse premium digital products, add them to your cart, and checkout seamlessly with real-time notifications.

## Features

- **Product Showcase**: Browse 6 premium digital tools with detailed descriptions and pricing
- **Shopping Cart**: Add/remove products with real-time cart count in navbar
- **Toast Notifications**: User-friendly notifications for all cart actions using React-Toastify
- **Responsive Design**: Mobile-first design that works on all screen sizes
- **Tag System**: Products tagged as "Best Seller", "Popular", or "New"
- **Feature Lists**: Detailed features for each product with checkmarks
- **Checkout System**: Complete checkout flow with total price calculation
- **Badge Counter**: Dynamic cart badge in navbar showing number of items

## Technologies Used

- **React.js** - UI library and component management
- **Tailwind CSS** - Utility-first CSS framework
- **DaisyUI** - Tailwind CSS component library
- **JavaScript (ES6+)** - Modern JavaScript features
- **React-Toastify (NPM Package)** - Toast notifications
- **JSON** - Product data management
- **Lucide React** - Icon library
- **Vite** - Fast build tool and dev server
